HTML5 Markup

HTML5 is the latest version of HyperText Markup Language used to structure and present content on the web. It introduces new semantic elements, multimedia support (audio and video), better graphics handling with canvas, and improved performance without relying on plugins.

CSS3 Styling

CSS (Cascading Style Sheets) is used to style and design web pages. It controls layout, colors, fonts, spacing, and responsiveness, helping make websites visually attractive and user-friendly.

Graphic Design

Graphic design is the art of creating visual content using images, colors, typography, and layout to communicate ideas effectively. It is commonly used in branding, advertising, websites, and social media.
